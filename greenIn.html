About the API being tested

The green electricity index shows when a lot of renewable electricity is in the grid and can be used in the household. This means that everyone can support the energy transition in Germany and help ensure that green electricity is expanded efficiently.

About this page

This page performs queries for several zip codes in Berlin, ever two hours daily. This page does not display results found through the GrünstromIndex API and focuses solely on time taken to perform queries.

More information about the GrünstromIndex API can be found here.
